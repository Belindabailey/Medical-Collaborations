Project overview

Two inventions. One justice gap closed.

Drug-facilitated sexual assault has a systematic evidentiary failure: perpetrators choose zolpidem because it clears the body in 7-8 hours. Rape kit backlogs run to 400,000 untested kits. Victims who were unconscious have no conscious memory to testify from.

These two provisional patents address both sides of that failure simultaneously — a victim-side biomarker triage system that reads the body's involuntary biological record, and a perpetrator-side neural retrieval system that reads the crime-specific memory directly from the perpetrator's brain under warrant.
